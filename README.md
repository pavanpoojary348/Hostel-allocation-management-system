# 🏠 Hostel Allocation Management System

## 📌 Project Description

The **Hostel Allocation Management System** is a full-stack web application designed to simplify and automate the process of hostel room allocation for students.

It allows students to register, log in, and manage their hostel details, while administrators can manage room allocation efficiently.

---

## 🚀 Technologies Used

### 🔹 Frontend

* React.js
* HTML, CSS, JavaScript

### 🔹 Backend

* Node.js
* Express.js

### 🔹 Database

* MySQL

---

## ✨ Features

### 👨‍🎓 Student

* Register account
* Login securely
* View dashboard
* Apply for hostel

### 🛠️ Admin (if implemented)

* Manage students
* Allocate rooms
* View requests

## ⚙️ Installation & Setup

### 1️⃣ Clone the repository

```
git clone https://github.com/pavanpoojary348/Hostel-allocation-management-system.git
```

### 2️⃣ Backend Setup

```
cd hostel-allocation/backend
npm install
npm start
```

### 3️⃣ Frontend Setup

```
cd hostel-allocation/frontend
npm install
npm start
```
